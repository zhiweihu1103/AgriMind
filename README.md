## 🚀 项目愿景

**让智能扎根农业，让数据驱动育种。**  
后稷农业大模型体系是由山西省后稷实验室、山西农业大学、山西大学联合提出，致力于构建**中文农业领域首个“通用农业+育种智能”双轮驱动的大模型生态**，贯通模型、数据、工具与评测，服务农业智能化与种业创新。
