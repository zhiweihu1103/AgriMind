> **让智能扎根农业，让数据驱动育种。**  
> 后稷农业大模型体系致力于构建**中文农业领域首个“通用农业+育种智能”双轮驱动的大模型生态**，贯通模型、数据、工具与评测，服务农业智能化与种业创新。

---

## 🚀 项目愿景

- 🌱 构建 **通用农业大模型底座**
- 🧬 打造 **面向育种的核心能力体系**
- 📚 构建 **高质量农业与育种指令数据**
- 🤖 推动 **农业智能体研发**
- 🌍 建立 **中文农业AI开源生态**

## 🧠 核心项目矩阵

### 🌾 通用农业大模型（Foundation Layer）

- 🔹 **AgriMind**  
  👉 首个中文开源农业大模型（农业知识理解与推理基础）  
  🔗 https://github.com/zhiweihu1103/AgriMa  

- 🔹 **AgriAgent**  
  👉 首个中文开源农业多模态大模型（文本 + 图像农业理解）  
  🔗 https://github.com/zhiweihu1103/AgriAgent  

---

### 📖 数据与指令体系（Data Layer）

- 🔹 **AgriInstruct**  
  👉 首个中文开源农业微调指令集（覆盖农业 + 育种任务）  
  🔗 https://github.com/zhiweihu1103/AgriInstruct
